How it works
1️⃣
Upload
Drop in your CommBank PDF or CSV statement. Everything stays in your browser — nothing is sent anywhere.
2️⃣
Review
Transactions are auto-categorised. Click any category pill to change it. Add cash or manual entries too.
3️⃣
Track
Set monthly budgets, add recurring fixed expenses, and upload next month's statement to see trends.

🔒 Your data never leaves your device. This tool runs entirely in your browser.

How to create a token
  1. Go to github.com → Settings → Developer settings
  2. Click Personal access tokens → Tokens (classic)
  3. Click Generate new token (classic)
  4. Name it finance-tracker, tick repo only
  5. Click Generate token → copy it

Export to Excel / Google Sheets

  1. Click Copy data below
  2. Open Excel or Google Sheets → click cell A1
  3. Press Ctrl+V (or Cmd+V on Mac)
  4. All transactions and the summary will paste in columns
